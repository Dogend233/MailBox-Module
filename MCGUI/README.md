# MCGUI  
MinecraftGUI: 用原版Minecraft的GUI收发邮件  
插件内的GUIPackage类可以方便从yml中读取一个基础的箱子GUI  
  
### 前置模块
[公共界面API - CommonGUIAPI](../CommonGUIAPI)
